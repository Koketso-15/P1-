## 🧪 Usage Examples

### Chat with AI
```
User: "Explain quantum computing in simple terms"
AI: [Responds with a clear explanation...]
```

### Analyze Data
```
User: [Uploads sales_data.csv]
User: "What's the total revenue by month?"
AI: [Generates a chart + summary...]
```

### Compare Models
```
Switch between Llama 3 (Groq) and DeepSeek (OpenRouter) 
to compare response quality for the same prompt.
```
